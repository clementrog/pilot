# LOG

## Format

```
[DATE] [EVENT] [TASK] [RESULT] [NOTES]
```

## Events

| Event | Meaning |
|-------|---------|
| `INIT` | Project initialized |
| `SPEC` | Task spec written |
| `BUILD` | Implementation done |
| `VERIFY_PASS` | Evidence approved |
| `VERIFY_FAIL` | Evidence rejected |
| `COMMIT` | Changes committed |
| `HEALTH_PASS` | Health check passed |
| `HEALTH_FAIL` | Health check failed |
| `ROLLBACK` | Reverted to LKG |
| `BLOCKED` | Task blocked |
